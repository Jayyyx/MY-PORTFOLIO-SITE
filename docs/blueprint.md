# **App Name**: Lavoe's Launchpad

## Core Features:

- Hero Section: Display of key information (name, contact details, location) prominently on the landing page.
- Education & Experience: Dedicated section to showcase educational background and relevant internship experience.
- Skills Showcase: Clear categorization and display of technical skills (strong and weak stacks).
- Projects display: Project cards, with links to your live apps and/or Github repos
- Aesthetic Comparator AI Tool: Visual link to Delly's Portfolio for visual inspiration. This will operate as a tool to enhance my ability to match design aesthetics with my portfolio site, using that external site as inspiration for generating improved and updated designs

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke calmness and trustworthiness.
- Background color: Very light gray (#F0F0F0) for a clean and professional look.
- Accent color: Dark blue (#00008B) to highlight key sections and call-to-action buttons.
- Body font: 'PT Sans' (sans-serif) for readability and a modern feel.
- Headline font: 'Space Grotesk' (sans-serif) for a tech-oriented feel.
- Use simple, professional icons for skills and contact information.
- Clean, well-structured layout with clear sections and visual hierarchy.
- Subtle animations for transitions and hover effects.
